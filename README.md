# UpsideLang Translator

Este repositório contém um tradutor de arquivos `.up` (UpsideLang) para Python implementado com [PLY](https://www.dabeaz.com/ply/) (Lex e Yacc em Python).

## Como usar

1. Instale as dependências:

```bash
pip install -r requirements.txt
```

2. Execute o tradutor indicando um arquivo `.up` de entrada e o destino `.py`:

```bash
python translator.py codigo.up codigo.py
```
